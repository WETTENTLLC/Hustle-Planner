# 📄 How to Print Your Flyer

## 🖨️ Quick Print Instructions:

### **Method 1: Direct Print (Easiest)**
1. Open `FLYER.html` in your browser (Chrome, Edge, Safari, etc.)
2. Press **Ctrl+P** (Windows) or **Cmd+P** (Mac)
3. Settings:
   - Paper: **Letter (8.5 x 11 inches)**
   - Margins: **None** or **Minimal**
   - Background graphics: **ON** ✅
4. Click **Print**

### **Method 2: Save as PDF First**
1. Open `FLYER.html` in browser
2. Press **Ctrl+P** (Windows) or **Cmd+P** (Mac)
3. Destination: **Save as PDF**
4. Save the PDF
5. Open PDF and print normally

## 📱 Adding a QR Code (IMPORTANT!)

### **Option 1: Free QR Code Generator**
1. Go to: https://www.qr-code-generator.com/
2. Select "URL" type
3. Paste: `https://wettentllc.github.io/Hustle-Planner/`
4. Click "Create QR Code"
5. Download as PNG
6. Edit FLYER.html and replace the "SCAN ME!" div with:
   ```html
   <img src="path-to-your-qr-code.png" style="width: 150px; height: 150px;">
   ```

## 🎨 Customization Tips:

### **Change Colors:**
Look for these in the HTML:

- `#a855f7` = Purple
- Change to match club theme if needed

### **Add Your Instagram/Contact:**
Add before the tear-off section:
```html

    Follow: @your_instagram | Text: 555-1234

### **Multiple Clubs:**
Add club name at the top:

    Available at [CLUB NAME]
</div>
```

## 📋 Printing Tips:

### **Paper Recommendations:**
- ✅ **Cardstock** (110 lb) - Best, most durable
- ✅ **Glossy paper** - Eye-catching
- ✅ **Regular paper** - Works fine, just not as fancy

### **Print Quality:**
- Set to **High Quality** or **Best**
- Make sure **Color** is selected (not B&W)
- Check **Background Graphics** is ON

### **Multiple Copies:**
- Print 5-10 at a time
- Hang in different locations
- Replace when worn out

## ✂️ Tear-Off Tabs:

The bottom has 8 tear-off tabs with the URL!

**How to Cut:**
1. Print the flyer
2. Cut along the dashed line
3. Cut between each tab (scissors or paper cutter)
4. Girls can tear off a tab to remember the URL

## 🎯 Pro Tips:

1. **Laminate It** - Lasts longer in humid dressing rooms
2. **Add Glitter** - Makes it stand out (spray adhesive + glitter)
3. **Use Magnets** - Stick to lockers without tape
4. **Multiple Colors** - Print on different colored paper
5. **Add Stickers** - Sparkly stickers around the edges

## 🔄 Alternative Designs:

### **Canva Template:**
If you want to design in Canva:
1. Create new design (8.5 x 11 inches)
2. Use same text from flyer
3. Add photos/graphics
4. Keep QR code & URL prominent

### **Vertical Half-Page:**
Print 2 per page to save paper:
1. Adjust flyer height to 5.5 inches
2. Print 2 copies on one page
3. Cut in half
4. Hang more places!
